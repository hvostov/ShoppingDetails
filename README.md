# C++ ShoppingDetails

Приложение для учета товаров, автоматизированного расчета цен, формирования отчетов.

## Возможности

- Загрузка данных из типового бланка заказов (Excel)
- Пересчет цен в рубли
- Расчет выручки

## Требования

- C++17 и выше.
- Qt 6
- CMake 3.2 и выше
